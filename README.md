## Overview

This repository serves as a learning resource and practice collection for data structures and algorithms. It includes implementations of core data structures, problem-solving exercises, and assignment solutions.

## Data Structures Covered

- **Arrays** - Basic array operations and problems
- **Linked Lists** - Singly and doubly linked list implementations
- **Binary Search Trees (BST)** - Tree traversal, insertion, deletion, and search operations
- **Trees** - General tree structures and tree-based algorithms
- **Graphs** - Adjacency matrix and adjacency list representations, graph algorithms
- **Heaps** - Min-heap and max-heap implementations
- **Stacks & Queues** - Stack and queue data structures with various operations
- **Hashing** - Hash table implementations and hash-based problems
- **Recursion** - Recursive algorithms and problem-solving techniques

## Getting Started

### Prerequisites

- Python 3.x

### Running the Code

1. Clone the repository:
```bash
git clone https://github.com/yourusername/Data-Structure-Codes.git
cd Data-Structure-Codes
```

2. Navigate to the specific data structure folder or file you want to explore:
```bash
# Example: Run a BST implementation
python BST.py

# Example: Run graph algorithms
python graph.py
```

3. Each file can be run independently:
```bash
python <filename>.py
```
